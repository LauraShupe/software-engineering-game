## Functional Requirements
1 is highest priority

| ID   | Requirement              | Priority | Reasoning                                                           |
|------|--------------------------|----------|---------------------------------------------------------------------|
| FR1  | Player Movement          | 1        | User must be able to control the character accurately               |
| FR2  | Input Validation         | 2        | Input must be tracked correctly and within a reasonable time        |
| FR3  | Resource Collection      | 3        | Game should collect the user resource collection accurately         |
| FR4  | Enemy Logic              | 4        | Enemy interacts with the user following the written logic           |
| FR5  | Win condition detection  | 5        | Able to identify when the user has won the game                     |
| FR6  | Level transition system  | 6        | Able to go between levels once user meets requirements              |
| FR7  | Login                    | 7        | User must be able to login with a username and password to track scores |
| FR8  | Replay/Reset Function    | 8        | Able to reset/replay level at user's request                        |
| FR9  | UI traversal             | 9        | User must be able to interact with the game easily and efficiently  |
| FR10 | Level Timer              | 10       | Timer so user can track speed                                       |


## Non-Functional Requirements
| ID   | Requirement     | Description                                               |
|------|-----------------|-----------------------------------------------------------|
| NFR1 | Performance     | Game must run smoothly                                    |
| NFR2 | Reliability     | Game must not crash with multiple playthroughs            |
| NFR3 | Usability       | Game must be easy to learn and navigate                   |
| NFR4 | Responsiveness  | Game must respond to user input within 5 seconds          |
| NFR5 | Accuracy        | Game must track the time and score correctly              |
